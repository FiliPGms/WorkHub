Sistema para a gestão de um espaço de
coworking denominado WorkHub. O "WorkHub" é um espaço de coworking que oferece diversos tipos de ambientes de trabalho para
profissionais, freelancers e pequenas equipes. Os clientes podem reservar estações de trabalho,
salas privadas, salas de reunião ou auditórios, por hora, além de contratar serviços adicionais como
café premium, lockers, estacionamento e recebimento de correspondências.
